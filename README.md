# Neural Network From Scratch

A machine learning project focused on building a neural network framework from scratch in Python and applying it to sign-language image classification. The project includes a custom automatic differentiation engine, manual backpropagation, and comparisons against scikit-learn baselines.

## Key Contributions
- Built a custom automatic differentiation engine using computational graphs
- Implemented neurons, feedforward layers, and a multilayer perceptron from scratch
- Implemented forward and backward propagation manually
- Built mini-batch stochastic gradient descent training with custom loss functions
- Applied PCA for dimensionality reduction and evaluated performance against sklearn models

## What I Learned
- Deep understanding of neural network internals and backpropagation
- How automatic differentiation frameworks operate under the hood
- Gradient-based optimization and training dynamics in machine learning
